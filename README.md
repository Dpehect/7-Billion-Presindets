## What is 7 Billion Presidents?

A movement to build a better future for all 7 billion people on Earth. Profits go to humanity, not shareholders. Climate stability, ecosystem restoration, and social projects.

## What's on the Site?

- Responsive navigation
- Hero section
- About us
- Vision & Mission
- Partners
- Newsletter form
- Footer

## Setup

```bash
npm install
npm run dev
```

Opens at `localhost:5173`.

## Build

```bash
npm run build
```

Creates a `dist` folder. Upload that to host it.

## Files

- `src/components/` - All components
- `src/App.css` - Styles
- `src/App.jsx` - Main app

## Where to Deploy?

Netlify, Vercel, GitHub Pages, or any hosting. Just upload the `dist` folder.
